# 本机 Python 服务：piano_dialogue_server

这里是本仓库的 **Python 工作区**，用于两件事：

1. **Piano Dialogue**：本机 AI 钢琴对话（Turn-based 生成/回放）
2. **OMR HTTP**：把 PDF/图片谱转换为 MusicXML（供 AVP 导入）

> 根目录 `README.md` 是当前对外的“最终文档入口”；这里提供面向开发/排障的最小索引，避免重复与漂移。

## 快速开始

请按根目录文档执行（首次安装、模型放置、启动服务、健康检查等）：

- `README.md` →「🤖 Piano Dialogue（AI 钢琴对话模式）」
- `README.md` →「🧠 本机 Python 服务（Dialogue + OMR HTTP）」

## OMR（PDF/图片 → MusicXML）

- CLI 转换：`README.md` →「🧾 OMR：PDF/图片 → MusicXML」→「A) CLI 转换」
